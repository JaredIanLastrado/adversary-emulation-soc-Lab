# 01 – Wazuh Alert Intake

## Overview

The Wazuh Alert Intake playbook automates the initial handling of security alerts generated by Wazuh.

Wazuh sends alert metadata to a Shuffle webhook. Shuffle evaluates the Wazuh rule level, routes the alert through the appropriate severity branch, creates an alert in TheHive, and posts a formatted notification to the AESOC Slack alert channel.

This provides a consistent starting point for analyst triage while reducing manual alert entry across multiple systems.
