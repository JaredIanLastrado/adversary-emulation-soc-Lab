# Security Onion Dashboards

This directory contains AESOC dashboards built in Security Onion for network security monitoring.

The dashboards summarize network alerts, connections, protocols, and activity observed through mirrored lab traffic.

## Current Dashboard

- [Network Threat Monitoring Dashboard](Dashboard-001-Network-Threat-Monitoring/)

The dashboard demonstrates:

- Network alert visualization
- Suricata alert monitoring
- Zeek network-metadata analysis
- Protocol and connection visibility
- SOC-focused network monitoring
