# Wazuh Dashboards

This directory contains AESOC dashboards built in Wazuh for host-based security monitoring.

The dashboards summarize endpoint alerts and security events collected from Windows and Linux systems.

## Current Dashboard

- [Security Monitoring Dashboard](Dashboard-001-Security-Monitoring/)

The dashboard demonstrates:

- Wazuh alert visualization
- Endpoint security-event monitoring
- Severity and rule-level analysis
- Host and agent visibility
- SOC-focused dashboard design
