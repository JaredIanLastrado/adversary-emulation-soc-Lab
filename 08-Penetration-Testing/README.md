# Rekall Corporation Penetration Test

An authorized, multi-platform penetration-testing assessment completed in a fictional cybersecurity bootcamp environment. The engagement evaluated a web application, Linux servers, and a Windows host through reconnaissance, vulnerability discovery, controlled exploitation, impact validation, risk analysis, and remediation planning.

> **Ethical-use notice:** This project documents activity performed only within an isolated, authorized training environment. No real client systems or production data were tested. Challenge flags, credentials, and unnecessary sensitive details were removed from the public portfolio edition.

## Assessment Summary

The assessment validated seven vulnerabilities across web, Linux, and Windows systems:

- Unrestricted file upload
- Operating system command injection
- Apache Tomcat remote code execution
- Apache Struts remote code execution
- Weak SSH credentials and Linux privilege escalation
- SLMail POP3 buffer overflow
- Sensitive data exposure

**Risk distribution:** 3 Critical, 3 High, and 1 Medium finding.

## Skills Demonstrated

- Reconnaissance and service enumeration
- Vulnerability scanning and manual validation
- Web application security testing
- Controlled exploitation of Linux and Windows systems
- Privilege-escalation validation
- Evidence collection and impact analysis
- Risk rating and remediation development
- MITRE ATT&CK mapping
- Professional penetration-test reporting

## Tools and Technologies

OSINT/WHOIS, Nmap, Nessus, Metasploit, SSH, Linux, Windows, Apache Tomcat, Apache Struts, and manual web-input testing.
